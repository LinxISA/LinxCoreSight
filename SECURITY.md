# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability in LinxCoreSight:

1. Do not open a public GitHub issue.
2. Prefer using GitHub Security Advisories (if enabled for the repository).
3. Otherwise, contact the maintainer via GitHub: `@zhoubot`.

Please include:

- a clear description of the issue and impact
- steps to reproduce
- affected versions/commit hash (if known)
- any proposed fix or mitigation

## Supported Versions

Only the latest release is supported with security fixes.

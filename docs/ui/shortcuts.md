# Trace Viewer Shortcuts

## Mouse
- `Wheel`: vertical scroll.
- `Shift + Wheel`: horizontal pan.
- `Ctrl/Cmd + Wheel`: zoom anchored at cursor.
- `Middle drag`: pan (x + y).
- `Space + Left drag`: pan (x + y).
- Drag splitter at left pane boundary: resize label pane.

## Keyboard Navigation
- `Arrow Up/Down` or `K/J`: row scroll.
- `Arrow Left/Right` or `H/L`: cycle pan.
- `+ / - / 0`: zoom in / zoom out / reset zoom.
- `Home / End`: jump to top-left / bottom.
- `PgUp / PgDn`: page scroll.

## Tabs
- `Ctrl/Cmd + Tab`: next trace tab.
- `Ctrl/Cmd + Shift + Tab`: previous trace tab.
- `Ctrl/Cmd + W`: close active trace tab.

# CoreMark Demo (Prepared Project)

This prepared project builds and runs CoreMark in LinxCoreSight.

## Usage in IDE

1. Click Build.
2. Click Run.
3. Confirm output contains `Correct operation validated.`

## Notes

- The build uses `core_list_join.c` at `-O0` and the rest at `-O2`.
- CoreMark trademark/reporting rules apply (see `third_party/benchmarks/coremark/LICENSE.md`).
